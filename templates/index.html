Powered by a UFPA student.

Universidade Federal do Pará (Federal University of Pará, in English) is positioned among the region's most respected public universities. Despite its problems, UFPA has been a knowledge-spreader in a historically poor area.

A Python of possibilities.

This website has been created through the employment of Python and Flask, with many moduled included to power its functions. It is an example of how dynamically-generated pages should be: simple, straightforward, beautiful.

We have an API!

Our implementation follows the instructions provided on project1 page of CS50's Web Programming with Python and JavaScript, which stipulates that "if users make a GET request to your website’s api route, where is an ISBN number, your website should return a JSON response containing the book’s title, author, publication date, ISBN number, review count, and average score."

Built with SQLAlchemy!

According to project1 page of CS50's Web Programming with Python and JavaScript, "users will be able to register for" books50web "and then log in using their username and password. Once they log in, they will be able to search for books, leave reviews for individual books, and see the reviews made by other people."
